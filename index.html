¿Qué es HTML?

¿Qué es HTML? HTML (HyperText Markup Language) es el lenguaje estándar que se utiliza para crear y diseñar páginas web. Utiliza una serie de etiquetas y atributos que definen la estructura y el contenido de una página web.

Beneficios de HTML

Simplicidad: Es fácil de aprender y usar.
Compatibilidad: Es compatible con todos los navegadores web.
Estandarización: Está estandarizado por el World Wide Web Consortium (W3C), asegurando que los documentos sean consistentes.
Flexibilidad: Permite la inclusión de otros lenguajes como CSS y JavaScript para mejorar el diseño y la funcionalidad.

Utilización de las Páginas Web

Las páginas web se utilizan para diversos propósitos, incluyendo:

Información: Difundir noticias, artículos, blogs, etc.
Comercio Electrónico: Vender productos y servicios.
Entretenimiento: Streaming de videos, juegos online, etc.
Comunicación: Redes sociales, foros, etc.
Educación: Cursos en línea, tutoriales, etc.

Diferencias entre Página Estática y Dinámica

Página Estática:

Contenido Fijo: El contenido no cambia a menos que el desarrollador lo modifique manualmente.
Ejemplo: Un sitio web de portafolio personal.
Tecnología: Principalmente HTML y CSS.

Página Dinámica:

Contenido Interactivo: El contenido puede cambiar en función de las interacciones del usuario o el tiempo.
Ejemplo: Redes sociales, sitios web de noticias.
Tecnología: HTML, CSS, JavaScript, bases de datos y lenguajes de servidor como PHP o Python.

Diferencia entre Página Web y Portal Web

Página Web:

Definición: Una única unidad o documento en la web.
Función: Proveer contenido específico sobre un tema.

Portal Web:

Definición: Un sitio web que sirve como punto de partida o entrada, ofreciendo enlaces a diferentes secciones de información.
Función: Actuar como un hub o centro de información para diversos recursos. Ejemplo: Yahoo!, MSN.

¿Qué es PHP y para qué se utiliza?

PHP es un lenguaje de programación de código abierto, muy popular y ampliamente utilizado en el desarrollo web. Se utiliza principalmente para crear páginas web dinámicas y se integra fácilmente con bases de datos como MySQL.

Programas necesarios para utilizar PHP

Para utilizar PHP, necesitas:

  • Un servidor web (por ejemplo, XAMPP)
  • PHP instalado en tu servidor
  • Un editor de texto (por ejemplo, Visual Studio Code)

¿Qué es un servidor web?

Un servidor web es un software que entrega páginas web a los usuarios a través de internet. Al recibir una solicitud de un navegador, el servidor web procesa la solicitud y envía los archivos solicitados al navegador.

¿Qué es MySQL y cuál es la diferencia entre SQL y MySQL?

MySQL es un sistema de gestión de bases de datos relacional (RDBMS) que utiliza SQL (Structured Query Language) para gestionar los datos. La diferencia principal es que SQL es un lenguaje de consulta utilizado para interactuar con bases de datos, mientras que MySQL es el software que implementa SQL para gestionar y almacenar datos.

¿Cómo se usa PHP directamente en una página web?

Puedes incrustar código PHP en un archivo HTML utilizando las etiquetas <?php ... ?>. Aquí tienes un ejemplo:

                                           <?php
                                               echo "Hola, mundo!";
                                           ?>

¿Qué son las variables y para qué sirven?

Las variables son contenedores que almacenan valores. Se utilizan para almacenar datos que se pueden manipular y utilizar posteriormente en un programa.

¿Qué es una variable string y qué función tiene?

Una variable string es una variable que almacena una secuencia de caracteres. Se utiliza para almacenar y manipular texto. Ejemplo:

                                           <?php
                                               $texto = "Esto es una cadena de texto";
                                               echo $texto;
                                           ?>

¿Cómo se usa la estructura if en PHP y para qué sirve?

La estructura if se utiliza para tomar decisiones en el código. Permite ejecutar un bloque de código si una condición es verdadera. Ejemplo:

                                           <?php
                                               $edad = 20;
                                               if ($edad >= 18) {
                                                   echo "Eres mayor de edad.";
                                               }
                                           ?>

Ciclos en PHP

Los bucles o repetidores se utilizan para ejecutar un bloque de código varias veces. En PHP, los bucles más comunes son for, while y do while.

                                           <?php
                                               // Bucle for
                                               for ($i = 0; $i < 10; $i++) {
                                                   echo $i;
                                               }
                                           ?>

Crear un formulario

Para crear un formulario en HTML:

                                           <form action="procesar.php" method="post">
                                               <label for="nombre">Nombre:</label>
                                               <input type="text" id="nombre" name="nombre"><br>
                                               <input type="submit" value="Enviar">
                                           </form>

Crear un formulario y guardar la información en un bloc de notas 

                                           <?php
                                               if ($_SERVER["REQUEST_METHOD"] == "POST") {
                                                   $nombre = $_POST["nombre"];
                                                   file_put_contents("datos.txt", $nombre . PHP_EOL, FILE_APPEND);
                                               }
                                           ?>
                                           <form action="" method="post">
                                               <label for="nombre">Nombre:</label>
                                               <input type="text" id="nombre" name="nombre"><br>
                                               <input type="submit" value="Enviar">
                                           </form>

                                               id INT AUTO_INCREMENT PRIMARY KEY,

Crear un formulario y almacenar la información en una base de datos 

                                           <?php
                                               if ($_SERVER["REQUEST_METHOD"] == "POST") {
                                                   $nombre = $_POST["nombre"];
                                                   $conexion = new mysqli("localhost", "usuario", "contraseña", "mi_base_de_datos");
                                                   $conexion->query("INSERT INTO usuarios (nombre) VALUES ('$nombre')");
                                                   $conexion->close();
                                               }
                                           ?>
                                           <form action="" method="post">
                                               <label for="nombre">Nombre:</label>
                                               <input type="text" id="nombre" name="nombre"><br>
                                               <input type="submit" value="Enviar">
                                           </form>

Crear un formulario para consultar información guardada en una base de datos 

                                           <?php
                                               if ($_SERVER["REQUEST_METHOD"] == "POST") {
                                                   $nombre = $_POST["nombre"];
                                                   $conexion = new mysqli("localhost", "usuario", "contraseña", "mi_base_de_datos");
                                                   $resultado = $conexion->query("SELECT * FROM usuarios WHERE nombre='$nombre'");
                                                   while ($fila = $resultado->fetch_assoc()) {
                                                       echo "ID: " . $fila["id"] . " - Nombre: " . $fila["nombre"] . "<br>";
                                                   }
                                                   $conexion->close();
                                               }
                                           ?>
                                           <form action="" method="post">
                                               <label for="nombre">Nombre:</label>
                                               <input type="text" id="nombre" name="nombre"><br>
                                               <input type="submit" value="Consultar">
                                           </form>
